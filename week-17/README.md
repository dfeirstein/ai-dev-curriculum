# Week 17: Advanced AI-Native Patterns

**Theme:** Level up your Claude Code mastery. Push the boundaries of what you can build.

You've been using Claude Code for 16 weeks. You know the basics cold: describe what you want, review the output, iterate, ship. But there's another level. This week you learn the advanced patterns that separate competent AI-native builders from exceptional ones.

You'll learn how to break down complex problems into sequential prompts, how to use Claude Code for code review and architecture analysis, how to navigate unfamiliar codebases, and how to contribute to the open source projects you've been using all along.

This week is about range. You're not just building your own projects anymore -- you're capable of understanding and contributing to any codebase.

---

## Learning Objectives

1. **Master complex prompting patterns** -- break large problems into effective sequences of prompts.
2. **Use Claude Code for code review** -- analyze code for bugs, improvements, and architectural issues.
3. **Navigate unfamiliar codebases** -- use Claude Code to understand libraries and projects you didn't build.
4. **Recognize and correct bad AI output** -- know when Claude gets it wrong and how to steer it back.
5. **Contribute to open source** or **publish a reusable package** -- extend your impact beyond your own projects.

## What You Ship

Either:
- **Option A:** A meaningful open source contribution (PR submitted to a real project), OR
- **Option B:** A published npm package extracted from your own code, with documentation and tests.

## Time Commitment

~12-15 hours across the week.

## Prerequisites

- **Weeks 1-16 completed.** You have multiple shipped projects and a mature Claude Code workflow.
- A GitHub account.
- For Option B: an npm account (free to create at npmjs.com).

## How to Approach This Week

1. **Read guide.md** thoroughly. This week introduces patterns you'll use for the rest of your career.
2. **Practice the patterns** on your existing projects before starting the main project.
3. **Choose Option A or Option B** based on what interests you more.
4. **Take your time with the research phase.** Understanding an unfamiliar codebase is the skill this week.

This week feels different from previous weeks. You're not building a new app from scratch. You're going deeper -- understanding other people's code, contributing to shared projects, and refining your most important tool: the ability to direct an AI effectively.
