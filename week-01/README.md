# Week 1: Your AI Development Environment

**Theme:** Meet your tools, build your first thing, put it on the internet.

## What You'll Learn

- How software development works in 2026 (hint: you don't write code)
- What each tool in your environment does and why it matters
- How to direct Claude Code to build software for you
- How to save your work with Git and put it on GitHub
- How to deploy a live website anyone can visit

## What You'll Ship

A **personal landing page** -- live on the internet, with your name, bio, and links. You'll share the URL with someone and watch their reaction.

This is real. It's not a tutorial exercise that disappears when you close the browser. It's yours, it's live, and it's the first thing you've ever built.

## Time Commitment

Expect to spend **8-10 hours** this week. Some of that is reading and understanding concepts. Most of it is hands-on work with Claude Code.

## Prerequisites

You must have completed **SETUP.md** before starting this week. That means:

- Ghostty is installed and working
- Zed is installed and working
- Claude Code CLI is installed and you can run `claude` in your terminal
- Git is installed
- GitHub CLI (`gh`) is installed and authenticated
- Node.js is installed
- You have a Vercel account

If any of these are missing, go back to SETUP.md first. Everything this week depends on your tools being ready.

## How to Approach This Week

1. **Read guide.md** from start to finish. Don't rush it. Understanding the concepts matters more than memorizing anything.
2. **Work through project.md** with Claude Code open. Follow each step. Experiment. If something doesn't work, that's fine -- you'll learn how to handle that too.
3. **Use resources.md** if you want to go deeper on any topic.

You're not expected to understand every file Claude Code creates. You're expected to understand what you asked for, whether you got it, and how to get it live on the internet. That's the job.

Let's go.
