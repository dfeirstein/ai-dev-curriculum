# The AI-Native Builder: Zero to Production in 20 Weeks

> You're not learning to code. You're learning to build software.

---

## What This Is

A 20-week hands-on curriculum that takes you from **never having built software** to **shipping production applications** — using AI as your builder.

You will never write code by hand. You will learn to **direct Claude Code** — an AI coding agent that lives in your terminal — to research, plan, and build software for you. Your job is to know *what* to build, *why* each tool exists, and *how* to guide the AI to produce production-quality output.

This is not a traditional coding bootcamp. Those teach you to be a typist. This teaches you to be a **builder**.

---

## The Mental Model

Think of it like filmmaking:

- **Traditional bootcamp** = Learning to operate a camera, adjust lighting, edit footage frame by frame
- **This curriculum** = Learning to be a director — you understand cinematography enough to give clear direction, but you're not operating the camera yourself

Your AI agent (Claude Code) is an incredibly capable crew. Your job is to:
1. **Research** — Understand the problem and the tools
2. **Plan** — Design the solution and architecture
3. **Execute** — Direct the AI to build it
4. **Verify** — Check the output with quality gates (lint, type-check, test, build)
5. **Ship** — Deploy to production

## How Each Week Works

Every week follows the same rhythm:

1. **Read the guide** — Understand the concepts (what tools do, why they matter)
2. **Direct Claude Code** — Research, plan, and build the week's project
3. **Verify with quality gates** — Lint, type-check, test, build
4. **Ship it** — Deploy, push, publish
5. **Check the acceptance criteria** — Did you build what was asked?

Each week has four files:
- `README.md` — Overview, objectives, what you ship
- `guide.md` — Concepts and directed workflow (NOT code tutorials)
- `project.md` — Project brief with acceptance criteria
- `resources.md` — Documentation links and references
