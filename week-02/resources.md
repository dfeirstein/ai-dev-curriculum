## Claude Code

- [CLAUDE.md Documentation](https://docs.anthropic.com/en/docs/claude-code/memory#claudemd) -- How CLAUDE.md files work, where they can live, and what to put in them.

- [Claude Code Overview](https://docs.anthropic.com/en/docs/claude-code/overview) -- Full documentation for Claude Code CLI features and usage.

## Git and Branching

- [Git Branching -- Branches in a Nutshell](https://git-scm.com/book/en/v2/Git-Branching-Branches-in-a-Nutshell) -- Visual explanation of how branches work. You do not need to memorize commands (Claude handles that), but understanding the mental model helps.
- [GitHub Pull Requests](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/about-pull-requests) -- What pull requests are and why they exist.
- [Atlassian Git Branch Tutorial](https://www.atlassian.com/git/tutorials/using-branches) -- Another clear explanation of branching with diagrams.

## Prompt Engineering

- [Anthropic Prompt Engineering Guide](https://docs.anthropic.com/en/docs/build-with-claude/prompt-engineering/overview) -- Official guide to writing effective prompts for Claude. Directly applicable to Claude Code.
- [Claude Code Best Practices](https://docs.anthropic.com/en/docs/claude-code/best-practices) -- Tips specific to getting the best results from Claude Code CLI.

## Deployment

- [Vercel Documentation](https://vercel.com/docs) -- How Vercel deployments work, custom domains, environment variables.
- [Vercel + Next.js Guide](https://vercel.com/docs/frameworks/nextjs) -- Specific guidance for deploying Next.js projects on Vercel.
